📐 Fórmula de horas extra

El valor de cada minuto extra se calcula así:
Sueldo base ÷ 30 ÷ 10 ÷ 60
Ese valor se multiplica por los minutos extra netos del trabajador (minutos después de su hora de salida, menos los minutos de tardanza acumulados en el mes). Es diferente por cada persona según su sueldo.

🔌 Conexión API de WhatsApp (envío 100% automático)

Si configuras la API oficial de WhatsApp Business (Meta Cloud API), los mensajes se enviarán solos, sin abrir WhatsApp ni tocar nada. Si dejas estos campos vacíos, el sistema abrirá WhatsApp Web con el mensaje listo (modo manual). Cómo obtener credenciales →
